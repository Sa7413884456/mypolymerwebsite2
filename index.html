Isotactic PMMA

Structure: All methyl ester (-COOCH₃) and methyl (-CH₃) groups positioned on the same side of the backbone

  • ✓ Semi-crystalline
  • ✓ Density: 1.23-1.26 g/cm³
  • ✓ Higher hardness
  • ✓ Tg: ~120°C

Atactic PMMA

Structure: Methyl groups randomly distributed on both sides of the polymer backbone

  • ✓ Amorphous (most common)
  • ✓ Density: 1.18-1.20 g/cm³
  • ✓ Better transparency
  • ✓ Tg: ~105°C

Syndiotactic PMMA

Structure: Methyl groups alternate regularly on opposite sides of the backbone

  • ✓ Semi-crystalline
  • ✓ Density: 1.24-1.27 g/cm³
  • ✓ High performance
  • ✓ Tg: ~115°C
